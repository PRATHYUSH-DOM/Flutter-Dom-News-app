
# Flutter Dom News App

Flutter news application created from scratch, Project includes - Lazy loading images, Menu Drawer, Provider, Skelton loading,

## Features

Lazy loading images

Menu Drawer

Skelton loading

Provider

API call from free news api [News API](https://newsapi.org/)
